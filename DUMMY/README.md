# DUMMY

DUMMY is a 4-axis robot arm and my first real mechatronics project. It focuses on the fundamentals of mechanical design, electrical circuitry, and software integration. While DUMMY serves no real-world purpose and was built primarily as a learning experience, the fact that it doubles as a cool tabletop robot arm definitely doesn't hurt. This section of the documentation might feel a bit unorganized since it will be iterated on quite a few times—or at least until I get bored of my mini "dumb-dumb."
