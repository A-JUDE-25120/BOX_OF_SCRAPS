###  4-Axis Robotic Arm
A custom-designed manipulator arms system utilizing gripper mechanism and computer vision integration.
* **The Blueprint:** Features real-time tracking logic, inverse kinematics implementations, and optimized joint torque curves.
* **State:** Tuning physical tolerances on the rack-and-pinion assembly.
